Este é um exemplo de como o **reStructuredText** funciona. 
Ele é muito usado no *Kernel do Linux* e em projetos Python.

Recursos Principais
-------------------

* **Extensível**: Aceita diretivas avançadas.
* **Padronizado**: Menos variações que o Markdown.
* **Técnico**: Ótimo para documentar APIs.
